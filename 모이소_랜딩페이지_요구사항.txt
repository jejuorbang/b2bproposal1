# 모이소 B2B 납품 랜딩페이지 제작 요구사항

## 📋 프로젝트 개요
**목적**: 제주 핸드메이드 소품 B2B 유통 사업 홍보 및 입점 문의 유도  
**타겟**: 카페, 숙박업소, 기념품샵 등 제주 지역 소매점 사업자

---

## 🎨 디자인 컨셉
- **색상**: 제주 감귤 오렌지, 하늘색, 파스텔 핑크/옐로우
- **스타일**: 밝고 경쾌한 MZ세대 감성, 모던하고 깔끔한 레이아웃
- **아이콘**: 감귤 🍊 모티브 적극 활용
- **반응형**: 모바일/태블릿/데스크톱 대응

---

## 📄 페이지 구조 및 섹션

### **1. Hero Section (메인 비주얼)**
```
- 캐치프레이즈: "제주 핸드메이드 소품 유통 제안서"
- 서브 카피: "3천가지 이상 제주도 소품을 온라인 한 번에!"
- CTA 버튼: "입점 문의하기", "카탈로그 보기"
- 배경: 핸드메이드 소품 이미지 (흐릿한 배경)
```

### **2. Introduction Section (모이소 소개)**
```
제목: "모이소(MOISO)란?"

콘텐츠:
- 3천가지 이상 제주도 소품 유통
- 제주도 최초 온라인 주문~결제 원-라운드 시스템
- 카테고리 아이콘 그리드:
  * 도자기, 패브릭, 캔들, 코바늘, DIY, 육아, 생활, 뜨개

시각 요소: 
- 모이소 매장 사진
- 카테고리별 제품 이미지 (원형 썸네일)
```

### **3. System Comparison Section (시스템 비교)**
```
제목: "온라인 유통 시스템 도입"

2단 비교 레이아웃:

[기존 시스템] vs [모이소 시스템]
- 번거로운 발품 판매 → 온라인 한 번에 주문
- 초보 운영 어려움 → 컨설팅 교육 진행
- 투자비용 큼 → 초기자본 절약
- 운영비 부담 → 단 한번에 포스등록 바코드 결제

차별화 포인트 강조:
✓ 원-라운드 시스템
✓ 컨설팅 지원
✓ 매장 브랜드 가치 창출
✓ 초기자본 절약
```

### **4. Product Value Section (제품 가치)**
```
제목: "늘 똑같은 선물 지겨우셨죠?"

4개 Value Proposition (카드 형식):

[지역성] 제주다움 개성 다양성
→ 친환경, 힐링 제품

[주체성] 끼와 재능을 마음껏 발휘
→ MZ 세대들이 찾는 브랜드 구축

[수익성] 간편한 샵인샵 창업
→ 추가 매출, 수익 창출

[혁신성] B2B 고도화 시스템
→ 손쉬운 온라인 결제 및 관리

하단 문구:
"제주문화와 정신이 담긴 제주 곳곳의 특별한 이야기 거리들을
3천가지 이상 핸드메이드 소품으로 다양하게 만나보실 수 있습니다."
```

### **5. Curation System Section (큐레이션)**
```
제목: "예술가와 함께하는 큐레이션 시스템"

3단계 프로세스 다이어그램:

[생산] → [제품] → [기획]
  ↓         ↓         ↓
핸드메이드 작가   선물하기 좋은   큐레이션을 통한
예술기획자      제주 소품      감성적 제품 기획
크리에이터

강점 리스트:
• 예술전공자의 전문 아트 큐레이팅
• 검증된 실력의 전문 예술가 300여명 소속
• 청년 예술가들의 참신한 스토리텔링
• 문화·관광·역사 전공자의 차별화된 기획력
```

### **6. Target Customer Section (타겟 고객)**
```
제목: "MZ세대들이 좋아하는 소품"

3개 키워드 (원형 강조):

[취향] 나만의 스타일에 맞게
     다양하고 재미있는 상품

[즐거움] 누군가를 위해, 또는 나를
       위해 설레는 마음으로 구매

[희소성] 다른곳에서 보지못한
       특별한 상품이 가득한 매장

하단 메시지:
"선물을 구매하는 사람도 행복하고, 
선물을 받게되는 사람도 행복해지는
젊은 고객이 찾아오는 매장을 만들어 보세요!"
```

### **7. Benefits Section (입점 효과)**
```
제목: "모이소 입점 시 얻을 수 있는 효과"

박스 강조 3가지:

1️⃣ 프랜차이즈 (제주 소품) 상품을 가맹비 없이 창업
2️⃣ 초보자도 자유롭게 개인창업 및 샵인샵, 부업 가능
3️⃣ 온라인으로 주문, 결제, 납품까지 원-라운드 운영

강조 문구:
"프랜차이즈의 장점을 그대로 
But, No 갤런티 No 가맹비"
```

### **8. Success Case Section (성공 사례)**
```
제목: "Shop in Shop 사례"

Before / After 비교:
- 이미지: 카페 매장 개선 전후 사진
- 그래프: 매출 100배 상승 (Before: 30만원 → After: 1000만원)

개선 포인트:
✓ SKU 보완
✓ 제품 품질 개선
✓ 객단가 상승
```

### **9. Process Section (납품 절차)**
```
제목: "유통 프로세스 안내"

4단계 플로우차트:

[1단계] 납품 상담 및 제품 검토
   ↓
[2단계] 거래서류 작성 및 등록
   ↓
[3단계] 상품 리스트 조율
   ↓
[4단계] 납품/판매
```

### **10. FAQ Section (자주 묻는 질문)**
```
제목: "자주 질문하는 Q&A"

아코디언 형식 5개 질문:

Q. 공급률은 어떻게 되나요?
A. 평균 공급률 60% (판매마진 40%, 부가세 포함)
   수제품: 60~70% / 공장제품: 50~60%
   100% 세금계산서 발행

Q. 주문하면 몇일 후 납품 가능할까요?
A. 1주 1회 제주도 동선별 납품 (동선 맞으면 1주 2회도 가능)

Q. 반품 가능할까요?
A. 납품 후 1~2개월 내 정상제품 반품 가능

Q. 결제는 어떻게 하나요?
A. 계좌이체, 카드결제 모두 가능 (반품시 역입금)

Q. 바코드 부착되어 있나요?
A. 바코드와 가격표 모두 부착됨
   포스 등록용 엑셀파일 제공
```
